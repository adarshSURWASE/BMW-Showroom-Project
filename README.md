# BMW Showroom Web Application 🚗

A Java Web Application for booking BMW cars online.

## Technologies Used
- Java (Servlet & JSP)
- MySQL Database
- HTML5
- CSS3
- JavaScript
- Apache Tomcat
- JDBC

## Features
- User Registration
- User Login
- Car Booking
- Cart System
- User Profile
- Logout System

## Project Structure
src/main/java → Java Servlets and utility classes  
src/main/webapp → HTML, JSP, CSS, JS files  
WEB-INF → Web configuration files  



## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat server
3. Create MySQL database
4. Run the project on server
