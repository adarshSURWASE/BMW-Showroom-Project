# BMW Showroom Web Application

This is a Java Web Application built using Servlets, JSP, and MySQL.

## Technologies Used
- Java
- Servlet
- JSP
- MySQL
- HTML
- CSS
- JavaScript
- Apache Tomcat

## Features
- User Registration
- User Login
- Car Booking
- Cart Management
- User Profile
- Logout System

## Project Structure
src/main/java → Java Servlets and utility classes  
src/main/webapp → JSP, HTML, CSS, JS files
